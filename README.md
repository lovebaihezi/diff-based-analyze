## Build from source

** Use Master Branch of Zig to build **

The `build.zig` will dynamic link the `LLVM`, `libgit2` and `libc`.

```
zig build -Doptimize=ReleaseSafe
```

Make sure you have `LLVM` and `libgit2` installed locally
